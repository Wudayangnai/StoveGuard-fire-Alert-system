StoveGuard Fire Alert System

Overview
StoveGuard is a compact and reliable fire detection and early warning system designed to reduce the risk of fire incidents in domestic kitchen environments.

The system continuously monitors the stove area using a combination of sensors to detect abnormal heat buildup and flame activity. By identifying early warning signs of fire-related hazards, StoveGuard provides timely alerts before situations escalate into serious accidents.

Users are instantly notified through:

 *Audible buzzer alerts
 *Real time web notifications

This ensures warnings are communicated effectively both inside the home and remotely.
